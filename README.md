#### przygotowanie środowiska pracy

1. zainstaluj poetry i dodaj do zmiennych środowiskowych
https://python-poetry.org/docs/#installing-with-the-official-installer

2. przy uzyciu poetry i pythona 3.12.8 utwórz wirtualne środowiskoa nastepnie zainstaluj zależności z pyproject.toml
```bash
poetry env use sciezka/do/pythona3.12.8/python.exe
poetry install
```
